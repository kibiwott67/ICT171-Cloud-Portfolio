About This Project

MedCare is hosted on a Linux (Ubuntu 22.04 LTS) virtual machine running on Microsoft Azure, configured and deployed as part of ICT171 at Murdoch University.

The server runs Apache2 with PHP, secured with an SSL/TLS certificate from Let's Encrypt, and is accessible at allankibiwott.net via GoDaddy DNS.

  • Ubuntu 22.04 LTS
  • Apache2 + PHP
  • Let's Encrypt SSL
  • Azure IaaS
  • GoDaddy DNS
